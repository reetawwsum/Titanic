# Titanic (Kaggle)
Titanic: Machine Learning from Disaster

## Features used in the model

1. Passenger class
2. Passenger sex
3. Sibsp
4. Parch
5. Fare
6. Embarked
7. Family size
8. Title in name
9. Age
10. Numeric/Alphanumeric ticket number
11. Number of digits in numeric ticket
12. Ticket type in alphanumeric ticket
